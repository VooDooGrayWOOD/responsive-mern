<h4 align="center">Client-server application (social blog) based on MongoDB, Express, React, Node js and Material UI. With jwt authentication (registration of new users),  add new post. Add & delete friends. All this when using Redux as a state management library. Two entities: users, posts.</h4>

<h3 align="center">Deploy -> https://responsive-mern.voodoograywood.ru/ </h3>
